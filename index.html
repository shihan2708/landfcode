HTML exercises

  1. Create a webpage that prints your name to the screen. [See solution]
  2. Create a webpage that prints the numbers 1 - 10 to the screen. [See solution]
  3. Create a webpage and set its title to "This is a webpage". [See solution]
  4. Create a webpage that prints the message "When was this webpage created? Check page's title for the answer." to the screen, and set the title of the page to the current date [See solution]
  5. Create a webpage that prints any text of your choosing to the screen, do not include a head section in the code. [See solution]

NOTE: Include comments in every HTML basics exercise.
